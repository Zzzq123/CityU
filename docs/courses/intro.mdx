## 🚀 **1. 如何使用课程指南**

按照以下步骤，轻松掌握课程与考试相关信息：

1. **📂 浏览课程分类**  
   在左侧侧边栏找到您的学院和系，轻松找到您感兴趣的课程。
2. **🔍 查看课程详情**  
   点击课程名称，即可获取课程简介、学习重点、评估方式和学生评价，为学习做足准备。
3. **📝 参考学生评价**  
   每个课程页面都有评价部分，借助往届学生的学习建议与考试心得，更高效地备考。
4. **✅ 根据需求选择课程**  
   结合职业目标与个人兴趣，制定最适合您的学习计划，为未来打下坚实基础。

## 📑 **2. 课程分类一览**

课程分类基于 **学院与系**，涵盖了所有 Master’s Programmes 的课程信息。

📂 **分层结构**
- **学院 (College)** 是课程分类的最高层级。
- 每个学院下再细分为具体的 **系 (Department)**。

📋 **快速访问**
- 通过侧边栏，您可以轻松访问所有课程分类，结构清晰，导航便捷。

## 💬 **5. 课程评价**
在每个课程页面，您可以查看同学们提交的真实反馈，为您的学习规划和备考提供有价值的参考。我们欢迎每一位同学积极参与课程评价，分享您的学习心得，与他人共同成长。

### **✏️ 如何参与评价？**

1️⃣ **课程页面底部评论**  
💬 在课程页面底部评论框直接提交您的评价：
- 我们会在审核后将评价内容收录到课程页面，并标注您的作者信息。

2️⃣ **GitHub 贡献**  
🌐 想更系统地分享您的课程体验？
- 请参考 [Contributing](https://penjc.github.io/CityU/contributing)，通过 GitHub 提交您的课程评价。

## 🌟 **6. 为课程指南贡献**

###  **📢 如何贡献？**

欢迎参考 [Contributing 指南](https://penjc.github.io/CityU/contributing)，为完善课程内容和丰富学生评价贡献您的力量：
- 📝 **课程信息**：包括课程简介、学习目标、评估方式等关键内容。
- ✍️ **课程评价**：分享您的课程体验，帮助其他同学更好地规划学习。

### 🎯 **为什么要贡献？**

✨ **助力学生成长**  
通过贡献更多内容，您将帮助更多同学获取全面、准确的课程信息，轻松规划学术和职业路径。

🌟 **支持开源社区**  
将课程指南打造成高质量的开源项目，让更多人从中受益，同时展现您的技术能力和贡献精神。

💡 **获得社区认可**  
成为 **CityU 手册**项目的一员，您的付出将被记录和感谢，为学术圈和开源社区提供重要价值。
