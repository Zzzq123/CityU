# 邵逸夫图书馆

[邵逸夫图书馆（Run Run Shaw Library）](https://www.cityu.edu.hk/lib/)是香港城市大学的重要学术资源中心，提供丰富的学习资源和服务，满足学生和教职员工的学术需求。

---

## 📖 图书馆简介

邵逸夫图书馆位于 **杨建文学术楼**（Yeung Kin Man Academic Building）内，拥有宽敞的学习空间、全面的馆藏资源以及先进的设备，旨在为学生提供一个理想的学习与研究环境。

## 📚 学习资源

- **馆藏资源**：
  - **印刷资料**：书籍、学术期刊、报纸等。
  - **电子资源**：电子书、数据库、学术期刊及多媒体资源。

- **在线服务**：
  - 文献查询与借阅服务。
  - 数据库访问和电子资源下载。

- **八校通（JULAC）证**：
  - 持证学生可访问香港其他大学的图书馆。

---

## 🖥️ 学习空间

- **自习区域**：
  - 提供安静的学习环境。
  - 配有个人座位和小组讨论室。

- **设施与设备**：
  - 电脑工作站和打印设备。
  - 茶水间供学生休息。

---

## 🛠️ 服务指南

- **入馆须知**：
  - 刷学生证入馆。
  - 禁止携带食物，但允许携带饮用水。

- **打印服务**：
  - 每位学生每年享有 1600 张免费打印额度。

- **咨询服务**：
  - 提供书目查询、文献推荐及使用培训。
