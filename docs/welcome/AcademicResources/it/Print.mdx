# 打印攻略

在城大，学生可以通过两种主要方式完成打印任务：使用校园电脑打印和网络上传打印。以下是详细的打印教程与相关资源分布指南，帮助您高效使用校园内的打印设备。

---

## 🖨️ 方法一：校园电脑打印

1. **登录校园电脑：**

在打印机附近的校园电脑上，使用您的 EID 登录。

**注意**：如上一位用户未登出，请务必重启电脑后再登录，避免文件上传至他人账户。

2. **获取文件：**

如果文件已保存在 Canvas 云盘，请点击桌面上的 **CityU 图标** 快捷键，选择 **Quicklink > Canvas**，找到文件并下载。

**提示**：不要直接预览打印，以免排版错误。

3. **打印文件：**

打开需要打印的文件，选择 **Print** 并设置打印选项（建议单份打印以确保质量）。

4. **开始打印：**

前往打印机，刷学生卡登录打印机界面。

选择 **Print/Print all** 以释放打印任务。

**提示**：使用学生卡打印默认为黑白模式。**彩色打印**需前往彩色打印机并刷八达通卡。

---

## 🌐 方法二：网络上传打印

1. **访问打印服务网站：**

确保您的设备连接校园网或通过 VPN 连接。

打开浏览器，输入以下网址并选择 **Advanced > Proceed**：
```
https://ccstung1.ad.cityu.edu.hk:9192/app
```
- 如果遇到“连接不是私密连接”提示，请选择继续访问或者键盘输入：
```
thisisunsafe
```

2. **登录网站：**

使用 EID 登录页面。

3. **上传文件：**

选择 **网络打印 > 提交任务 > 上传文件**。

4. **释放打印任务：**

确保文件状态为 **Held in a queue**。

前往任意打印机，刷学生卡完成打印。
